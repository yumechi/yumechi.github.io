# Create Blog Page

読んだ記事やスライドのリンクと感想を書くブログ記事を作成します。

## 実行手順

### 1. ユーザーから情報を収集

AskUserQuestion ツールを使って以下の情報を収集してください：

- **記事/スライドのタイトル**: 読んだ資料のタイトル
- **URL**: 資料へのリンク
- **タグ**: 記事のタグ（カンマ区切り、任意）
- **感想・学び**: 感想や学んだことの内容

### 2. ブログ記事の作成

以下の形式でファイルを作成します：

- **ファイルパス**: `blog/YYYY/MM-DD-slug/index.md`
  - `YYYY` は年（例: 2026）
  - `MM-DD` は月日（例: 01-30）
  - `slug` はタイトルから英数字とハイフンのみで生成（日本語の場合はローマ字か英訳）
- **例**: `blog/2026/01-30-react-hooks/index.md`

### 3. テンプレート

`blog/_template.md` を参考に、以下の形式で作成：

```markdown
---
slug: /YYYY-MM-DD-slug
title: 記事タイトル
authors: yumechi
tags: [tag1, tag2]
---

## 資料リンク

- [資料タイトル](URL)

## 感想・学び

感想や学びの内容をここに記載。
```

### 4. 作成後の確認

ファイル作成後、ユーザーに以下を確認：
- 作成したファイルのパス
- 内容のプレビュー
- 必要に応じて修正するか確認

## 注意事項

- slugは英数字とハイフンのみを使用
- 日本語タイトルの場合は適切な英語slugを生成
- タグは既存のブログ記事で使われているタグを参考にする
- 感想が長い場合は適切に段落分けする
