# Create Blog Page

読んだ記事やスライドのリンクと感想を書くブログ記事を作成します。

## 実行手順

### 1. ユーザーから情報を収集

以下の情報をユーザーに直接質問して収集してください：

- **ブログ記事のタイトル**: ブログページに表示されるタイトル
- **slug**: URLに使用される識別子（英数字とハイフンのみ）
- **タグ**: 記事のタグ（カンマ区切り、任意）
- **日付**: ファイルパスに使用する日付（デフォルト: 当日）
- **資料のタイトル**: 読んだ記事/スライドのタイトル
- **資料のURL**: 資料へのリンク
- **感想・学び**: 感想や学んだことの内容

### 2. ブログ記事の作成

以下の形式でファイルを作成します：

- **ファイルパス**: `blog/YYYY/MM-DD-slug/index.md`
  - `YYYY` は年（例: 2026）
  - `MM-DD` は月日（例: 01-30）
  - `slug` はユーザーが指定した値を使用
- **例**: `blog/2026/01-30-react-hooks/index.md`

### 3. テンプレート

`blog/_template.md` を参考に、以下の形式で作成：

```markdown
---
slug: /slug-name
title: ブログ記事のタイトル
authors: yumechi
tags: [tag1, tag2]
---

資料の概要や導入文（1-2文）

<!-- truncate -->

## 資料リンク

- [資料タイトル](URL)

## 感想・学び

感想や学びの内容をここに記載。
```

### 4. 作成後の確認

ファイル作成後、ユーザーに以下を確認：
- 作成したファイルのパス
- 内容のプレビュー
- 必要に応じて修正するか確認

## 注意事項

- slugは英数字とハイフンのみを使用
- タグは既存のブログ記事で使われているタグを参考にする
- 感想が長い場合は適切に段落分けする
- 日付を指定しない場合は当日の日付を使用する
